# 📚 Kütüphane Otomasyon Sistemi (Windows Forms - C#)

Modern C# Windows Forms teknolojisi ile geliştirilmiş bir **Kütüphane Otomasyon Sistemidir**.  
Kullanıcıların kitap işlemlerini kolayca yapabilmesini ve yöneticilerin sistemi kontrol edebilmesini sağlar.

---

## 🎯 Proje Amacı

Bu proje, kütüphane işlemlerini dijital ortama taşıyarak kullanıcıların kitap ödünç alma, iade ve takip işlemlerini daha hızlı ve kolay yapmasını sağlamak için geliştirilmiştir.

---

## 🚀 Proje Hakkında

Bu sistem ile kullanıcılar:
- Kayıt olabilir ve giriş yapabilir
- Kitapları ödünç alabilir ve iade edebilir
- Kendi profillerini yönetebilir

Yöneticiler ise:
- Kullanıcıları görüntüleyebilir
- Kitap durumlarını takip edebilir
- Geciken kitapları ve cezaları görebilir
- Sistemi genel olarak yönetebilir

---

## ✨ Sistem Özellikleri (Genel Bakış)

- 👤 Kullanıcı kayıt & giriş sistemi  
- 🔐 Admin ve normal kullanıcı ayrımı  
- 📖 Kitap ödünç alma & iade sistemi  
- ⏳ Gecikme tespiti ve ceza hesaplama  
- 🔍 Arama ve kategori filtreleme  
- 📜 İşlem geçmişi (log sistemi)  
- 👤 Profil sayfası  
- 🌙 Karanlık mod (Dark Mode)  
- 💾 JSON ile veri kaydetme  

---

## 🛠️ Kullanılan Teknolojiler

- C#
- Windows Forms
- .NET Framework
- JSON Serialization
- LINQ

---

## ▶️ Nasıl Çalıştırılır

1. Projeyi bilgisayarınıza indirin:
   - GitHub üzerinden **Code → Download ZIP** yapabilir veya
   - `git clone` komutu ile projeyi çekebilirsiniz.

2. Visual Studio ile açın:
   - `KutuphaneOtomasyon.sln` dosyasını açın.

3. Gerekli .NET Framework yüklü olduğundan emin olun:
   - Windows Forms uygulaması olduğu için .NET Framework 4.x gereklidir.

4. Projeyi çalıştırın:
   - `Start (F5)` tuşuna basarak uygulamayı başlatabilirsiniz.

5. (İlk kullanım)
   - Varsayılan admin hesabı:
     - Kullanıcı Adı: `admin`
     - Şifre: `admin`

--- 

## ⚠️ Notlar

- Proje Windows işletim sisteminde Visual Studio ile çalıştırılmalıdır.  
- Veriler JSON formatında saklanır (`kutuphane_verisi.json`).  
- İlk çalıştırmada veri dosyası otomatik oluşturulur.
